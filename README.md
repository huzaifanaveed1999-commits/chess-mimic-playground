# Chess Mimic Playground

An interactive Chess AI Playground designed to replicate specialized styles and prevent blunders, equipped with a 4-ply Stockfish lookahead, self-analysis tools, and custom visualization.

Developed using Flask, PyTorch, Stockfish, and customized client-side PGN processing.
